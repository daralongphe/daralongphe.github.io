PROJECTS:

Wordle Plus 2022

As a final project for the Software Construction class (CS 35L), I have recreated a prototype of the world famous "WORDLE" web application (officially run by The New York Times website). For the prototype, I implemented upon the original "WORDLE" app additional features that allow players to play the game with either four, five or six letters WORDLE and as many time per day as they would like. The project was supervised by Dr. Paul Eggert.

Link to Wordle Plus and see the code on GitHub

Line Following Arduino Car 2021

In the Intro to Electrical Engineering class's lab (ECE 3), I and my teammate were provided with a TI-RSLK car to which we have partially reconfigured by fusing in phototransistors as sensors to detect the changes in light reflected from the surface of the customized line track. We also reprogrammed the sensor fused car to allow autotnomous navigation through the line track using Arduino software. The project was supervised by Dr. Dennise M. Briggs.

Link to the demonstration video of the line following arduino car project and see the code on GitHub
